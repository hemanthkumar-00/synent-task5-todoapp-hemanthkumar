# To-Do List Web App

# Live Demo
 https://hemanthkumar-00.github.io/synent-task5-todoapp-hemanthkumar/
 
## Objective

To develop a task management application that allows users to add, delete, and mark tasks as completed, while ensuring tasks persist even after refreshing the page.

## Steps Performed

1. Designed the user interface using HTML and CSS.
2. Implemented task addition, deletion, and completion features using JavaScript.
3. Used localStorage to save tasks permanently in the browser.
4. Tested the application to ensure proper functionality after page refresh.

## Tools Used

* HTML5
* CSS3
* JavaScript
* VS Code
* Git & GitHub

## Outcome

A fully functional To-Do List application was developed successfully. Users can efficiently manage their daily tasks, and the application retains data using localStorage.
